# LAB_pandas_data_analysis

This repository contains a collection of data analysis exercises and projects using the pandas library in Python. It is designed to demonstrate data cleaning, exploration, and visualization techniques using realistic datasets.  

## Repo Structure
`data/`: Contains the raw datasets used for analysis.

`notebook/`: Jupyter Notebooks containing the step-by-step data analysis workflow.

`outputs/`: Generated files such as cleaned datasets, summary tables, or exported charts.

`Project Notes/`: Learning notes for Pandas library, including tricks and key caveats.
